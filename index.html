Tic-Tac-Toe

Tic Tac Toe is a simple and classic two-player game that is played on a 3x3 grid. One player uses “X” and the other uses “O”, and they take turns marking the empty cells in the grid. The goal is to be the first to place three of their marks in a row, either horizontally, vertically, or diagonally. If all nine cells are filled and neither player has achieved this, the game ends in a draw. This game is an excellent way to practice basic programming and logical thinking skills, often implemented using HTML, CSS, and JavaScript. It includes features like real-time player turns, win/draw detection, and a reset option to replay. Despite its simplicity, it provides a great introduction to interactive web-based game development.

Tech: HTML, CSS, JavaScript

Bill Calculator🧾

A Bill Calculator App built using HTML, CSS, and JavaScript allows users to enter items with their names, prices, and quantities,discount and tax, then calculates the total bill. The app displays the subtotal, applies any entered discount, and calculates the tax based on the provided percentages. The user can see the final total after applying the discount and adding the tax. The app provides an interactive list of added items, where users can view the price and quantity of each item. It also includes a clear function to reset the form, bill generation in the form of pdf storing of previous bills and a responsive design to ensure it works well on both mobile and desktop devices. The logic is handled by JavaScript, which dynamically updates the bill calculation and UI as the user interacts with the app.

Tech: HTML, CSS, JavaScript

Stopwath⏱️

A Stopwatch Timer is a simple and interactive web application built using HTML, CSS, and JavaScript that allows users to measure elapsed time with precision. It typically features core functionalities like Start, Stop, and Reset. When the "Start" button is clicked, the timer begins counting seconds (and optionally minutes and hours) in real-time. The "Stop" button pauses the timer, allowing the user to resume from where they left off, and the "Reset" button clears the time back to zero. Visually, it includes a clean display of the timer, responsive buttons, and often stylish transitions or animations for better user experience. This project is ideal for beginners to learn concepts like event handling, intervals (setInterval/clearInterval), DOM manipulation, and updating UI dynamically through JavaScript.

Tech: HTML, CSS, JavaScript
